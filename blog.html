Términos Clave

HTML
Lenguaje de marcado utilizado para estructurar páginas web.
CSS
Lenguaje de estilos utilizado para diseñar la apariencia visual de un sitio web.
JavaScript
Lenguaje de programación que permite agregar interacción y dinamismo a las páginas web.
Blog
Sitio web donde una persona o grupo publica contenido regularmente.
Responsive
Diseño web que se adapta a diferentes tamaños de pantalla como móviles, tabletas y computadoras.
